😶‍🌫️C++基础知识复习

在大一的时候我就已经学习过C#了，在此看来很多的知识都是相通的，所以在此就以C#与C++来对比，这样就更清晰，方便了。

😊基础

👦🏻首先C#是一种完全面向对象的语言，C++是既面向过程又面向对象的语言

C#中输出使用console.writeline（）（忽略大小写） 输入使用的是console.readline（）

C++中输出输入分别是cout<< <<endl;和cin>> >>endl;endl表示换行也可用\n。
“<<”是预定义的插入符，作用在流类对象cout上便可以实现项标准输出设备输出。
cout << 表达式 << 表达式…
标准输入是将提取符作用在流类对象cin上。
cin >> 表达式 >> 表达式…
提取符可以连续写多个，每个后面跟一个表达式，该表达式通常是用于存放输入值的变量。例如：
int a, b;
cin >> a >> b;

C++中的标识符的规则与C#是一样的不必多说，同样不用说的还有数据类型和变量（最开始引入的变量一定要赋值否者将是一个随意的无比大的值）。

C++中常量 

const float PI = 3.1415926；

也不必多说
看个例子基本上都可以明白了： 

 int radius;    

 cout<<"Please enter the radius!\n";

 cin>>radius;   

 cout<<“The radius is:”<<radius<<‘\n’;   

    cout<<“PI is:”<<3.14<<‘\n’;

 cout<<"Please enter a different radius!\n";

 cin>>radius;   

 cout<<"Now the radius is changed to:" 

         <<radius<<‘\n’;

 return 0;

😅进阶

👧🏻C++中的运算与C#中也基本一致，还有！，&&，||也一样，不必多说。

下面是C++中特有的条件运算：
表达式1？表达式2：表达式3（先求解表达式1，若表达式1的值为true，则求解表达式2，表达式2的值为最终结果，若表达式1的值为false，则求解表达式3，表达式3的值为最终结果）

🧑🏻然后是Sizeof运算和位运算，至于位运算我也没弄明白，据说只要记个笔记就可以了，那就不多说了，Sizeof运算就是查看“类型名”所指定的类型，或“表达式”的结果类型所占的字节数。

👨🏻接下来看看重点的数据类型转换；

l 将一个非布尔类型的算术值赋给布尔类型时，算术值为0则结果为false，否则结果为true。

l 将一个布尔值赋给非布尔类型时，布尔值为false则结果为0，布尔值为true则结果为1

l 将一个浮点数赋给整数类型时，结果值将只保留浮点数中的整数部分，小数部分将丢失。

l 将一个整数值赋给浮点类型时，小数部分记为0。如果整数所占的空间超过了浮点类型的容量，精度可能有损失。

混合运算时数据类型的转换——显式转换

显式类型转换的作用是将表达式的结果类型转换为类型说明符所指定的类型。

l 语法形式

类型说明符(表达式)

(类型说明符)表达式

类型转换操作符<类型说明符>(表达式)

类型转换操作符可以是：
const_cast、dynamic_cast、reinterpret_cast、static_cast

例：int(z), (int)z, static_cast(z)

👩🏻然后是选择结构和循环结构
放几个例子应该就可以懂了不必多说，和C#基本一样，需要注意的就是这里的各个结构中都是没有{}的，这就需要每个结构中的内容对齐来分段。
if就不放例子了，看下switch：

 int day;

 cin >> day;

 switch (day) {

 case 0: cout << "Sunday" << endl; break;

 case 1: cout << "Monday" << endl; break;

 case 2: cout << "Tuesday" << endl; break;

 case 3: cout << "Wednesday" << endl; break;

 default:

    cout<<"Day out of range Sunday .. Saturday"<< endl;   break;

while和for也基本一样，这里不必多说，就是要区分一下break和continue，
break是直接中断循环执行下一语句，continue是中断这一次循环进行下一次循环。

👶🏻然后是自定义类型，如下；
typedef 已有类型名 新类型名表

例： 

typedef double Area, Volume;  
typedef int Natural;
Natural i1,i2;
Area a;
Volume v;

using 新类型名 = 已有类型名;

例： 

using Area = double;
using Volume = double;
